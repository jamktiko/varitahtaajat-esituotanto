# Väri Tähtääjä

This is a game about colors for HT00CQ58-3003 Ohjelmistoprojekti 1

## Description

This game consists of challenges about color and short term memory about colors

## Getting Started

### Dependencies

- Svelte
- Typescript
- Netlify

### Installing

1. Clone this repository
2. Cd into the cloned repository folder
3. Run `npm install` to install dependencies
4. Run `npm run dev` to launch the development enviroment

### Executing program

Run `npm run dev` to start the enviroment
