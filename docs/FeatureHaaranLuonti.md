# Uuden feature haaran luominen

1. Siirry takaisin git repon kehitys haaraan `git checkout development`
2. Päivitä haara uusimpiin tietoihin. `git pull`
3. Luo uusi feature haara `git checkout -b feature/nimi-tähän`

- haaran nimessä ei saa olla välilyöntejä tai ÖÄÅ. Välilyönnit korvataan viivalla
