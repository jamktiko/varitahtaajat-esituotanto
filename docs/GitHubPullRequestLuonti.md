# Github pull requestin luonti

0. Avaa Repo selaimessa
1. Valitse haara dropdownista
   ![Haara Dropdown](images/HaaraDropDown.png)
2. Valitse joko Contribute -> Open pull reguest tai Compare & Pull Request
3. Vahvista Base on development
4. Valitse Katselioiksi vähintään yksi toinen tiimiläinen
5. Muokkaa otsikko ja kuvaus halueamaksisi.
6. Paina Create pull reguest

# Github Pull Reguestion Mergeäminen

1. Paina Merge Request kun on tarpeeksi hyväksyttäviä
2. Valitse Commit and merge. Tämä on että sitä committi historia ei katoa
